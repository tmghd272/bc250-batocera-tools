# BC-250 Batocera Tools

BC-250 scripts and tools designed to run within Batocera systems.

Batocera is a read-only system, so most parts of these scripts do not directly affect the kernel. Instead, they use Batocera services as a workaround to automatically load and execute scripts.

> **Note:** Only run this on BC-250 systems.
